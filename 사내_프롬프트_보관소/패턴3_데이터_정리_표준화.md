# 🔑 [패턴 3] 데이터 정리·표준화 프롬프트
이 프롬프트는 현장 반장님들의 거칠고 뒤섞인 점검 기록 메모, 오타 가득한 비정형 텍스트 데이터를 엑셀에 한 방에 복사-붙여넣기 할 수 있는 완벽한 정형 표(Table) 형태로 변환할 때 사용합니다.

---

## ✍️ 프롬프트 복사용 템플릿

```text
당신은 비정형 텍스트 데이터를 분석하여 정형 데이터베이스 표준 표로 재가공하는 데이터 엔지니어링 전문가입니다.

다음은 현장에서 메신저나 수첩 기록으로 전달받은 어수선하고 오타가 섞인 비정형 점검 메모 데이터입니다. 
이를 분석하여 엑셀에 바로 입력할 수 있도록 깔끔한 표 형식으로 정리해주세요.

[비정형 원본 데이터]
3/15 OO배수지, 김반장. 외벽 균열 3곳 폭 0.5mm 정도, 옥상 방수 들뜸
3/15 △△가압장 이반장 펌프1 정상 펌프2 진동 약간 베어링 점검필요
3/16 ㅇㅇ정수장. 박과장. 약품투입기 노후 부품 교체요망 염소투입 정상

출력 표 형식:
| 날짜 | 시설 | 점검자 | 항목 | 상태 | 조치 필요 여부 및 세부내용 |

요구사항:
1. 문맥을 파악하여 오타나 축약어(예: ㅇㅇ정수장 ➡️ OO정수장)를 바르게 표준화하여 표를 구성할 것
2. 한 행에 여러 개의 검출 항목(예: 균열과 방수 들뜸)이 섞여 있는 경우, 엑셀 필터링과 데이터 관리가 용이하도록 행을 분리하여 개별 레코드(Row)로 세분화해줄 것
3. 결과를 마크다운 표 형식으로 1차 출력하고, 엑셀에 복사해 갈 수 있도록 탭 구분자(TSV) 형식으로도 추가 제공할 것
```

## 🔮 황룡 대리의 튜닝 팁
*   이 프롬프트는 현장의 거친 텍스트를 데이터 자산으로 변환해 주는 마법입니다. **"행 분리"** 규칙을 요구사항 2번에 넣어두었기 때문에, AI가 알아서 한 줄 메모 속에 있던 '외벽 균열'과 '방수 들뜸'을 2개의 행으로 깔끔하게 찢어 엑셀 관리가 100배 편해집니다!
