# 🔑 [패턴 2] VBA·매크로 요청 프롬프트
이 프롬프트는 수많은 반복 점검 시트나 하위 데이터 시트에서 필요한 핵심 데이터만 쏙쏙 긁어모아 단 한 줄로 집계 시트에 정렬하는 '자동화 매크로 코드'를 AI로부터 안전하게 뽑아낼 때 사용합니다.

---

## ✍️ 프롬프트 복사용 템플릿

```text
당신은 엑셀 VBA 및 오피스 자동화 15년 차 최고의 프로그래머입니다.

목적: 매월 100개 이상의 점검 시트(시트명 형식: 점검_YYYYMMDD)에서 특정 정보를 추출해 "월간집계" 시트에 한 행씩 자동 누적 정리해주는 VBA 매크로 코드를 작성해주세요.

각 시트에서 추출할 셀 위치와 데이터 정보:
- B3 셀: 점검일
- C5 셀: 점검자
- D10 셀: 이상유무
- E15~E25 범위: 결함 내역

요구사항:
1. VBA 코드 내의 모든 변수 선언과 핵심 로직 단계마다 친절하고 완벽한 한국어 주석을 작성해주세요.
2. 속도 최적화를 위해 화면 업데이트 끄기(`Application.ScreenUpdating = False`) 설정을 코드 시작과 끝에 포함해주세요.
3. 비전공자 직원을 위해, 엑셀에서 [Alt + F11]을 눌러 모듈을 삽입하고 매크로를 붙여넣어 실행하는 단계별 매뉴얼을 텍스트로 친절히 함께 안내해주세요.
```

---

## 🔮 황룡 대리의 튜닝 팁
*   VBA는 엑셀 자동화의 꽃입니다! 만약 특정 셀이 아닌 특정 범위(예: 결함 내역 E15~E25)에서 데이터가 있는 행만 골라내고 싶다면, 요구사항에 **"데이터가 빈 칸이 아닌 경우에만 쉼표로 연결해서 한 셀에 넣어주세요"**라는 세부 규칙을 명시하면 훨씬 깔끔하게 데이터가 집계됩니다!
