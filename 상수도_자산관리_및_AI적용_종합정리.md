## 1. 상수도 자산관리 시스템 개요

### 정의

상수도 자산관리 시스템(Water Asset Management System)은 관로, 펌프, 정수장, 배수지 등 상수도 시설물을 **생애주기(Life Cycle) 관점**에서 체계적으로 관리하여 최소 비용으로 최적의 서비스 수준을 유지하기 위한 통합 관리 체계.

단순한 시설 대장 관리를 넘어, 노후도 평가·위험도 분석·투자 우선순위 결정까지 포괄하는 의사결정 지원 시스템.

### 핵심 구성 요소

| 구성요소 | 내용 |
|---------|------|
| 자산 인벤토리 | GIS 기반 모든 자산의 위치·재질·구경·매설연도·시공정보 DB화 |
| 상태평가 | 물리적·기능적·환경적 상태 종합 평가 |
| 위험도 분석 | 파손확률(LoF) × 파손결과(CoF) |
| 생애주기 비용 분석 (LCCA) | 초기투자비 + 운영유지비 + 갱신비용 + 폐기비용을 현재가치로 환산 |
| 투자 우선순위 결정 | 한정된 예산 내 위험도-비용효율 기반 사업 선정 |

### 국내 제도적 배경

- 2020년 「수도법」 개정으로 **상수도 자산관리 기본계획 수립** 의무화
- 환경부 「상수도 자산관리 업무 매뉴얼」을 통한 시행 절차 표준화
- ISO 55000(자산관리 국제표준) 및 PAS 55 체계 참고

### 최신 트렌드

- 머신러닝 기반 파손예측 (LSTM, XGBoost)
- 디지털 트윈을 활용한 실시간 자산상태 모니터링
- 위성·드론 영상을 통한 누수 탐지
- 미국 EPA의 **EPANET 3.0**
- 영국 UKWIR의 **Common Framework for Capital Maintenance Planning**

## 2. 자산관리 수행절차

환경부 「상수도 자산관리 업무 매뉴얼」과 ISO 55000 체계 기준.

### 1단계: 자산관리 계획 수립 (Planning)

조직의 자산관리 목표와 **서비스 수준(LoS, Level of Service)** 설정.
- 예: "유수율 95% 이상 유지", "관 파손 민원 연 50건 이하", "수압 1.5kgf/cm² 이상 보장"

### 2단계: 자산 현황 조사 및 DB 구축

GIS 기반 시설물 정보 수집·전산화.
- **관로**: 위치, 재질, 구경, 매설연도, 매설심도, 시공방법
- **시설물**: 펌프장, 배수지, 가압장의 제원 및 운영이력
- **부속설비**: 밸브, 소화전, 유량계, 압력계

⚠️ 가장 큰 과제: 데이터 누락·오류 검증, QGIS 속성테이블 정비

### 3단계: 상태평가 (Condition Assessment)

물리적·기능적·환경적 상태를 3~5등급으로 평가.
- **직접조사**: 내시경, 누수탐사, 비파괴검사, 시료채취
- **간접평가**: 파손이력, 민원이력, 수질이력, 수압이력 분석
- **통계분석**: 관종별·연령별 파손률 곡선(Bathtub curve) 도출

### 4단계: 위험도 평가 (Risk Assessment)

**위험도 = 파손확률(LoF) × 파손영향(CoF)**

- **LoF**: 관 나이, 재질, 토양 부식성, 과거 파손이력 → 회귀·머신러닝 모델
- **CoF**: 단수 영향인구, 중요시설(병원·학교·산업단지) 인접성, 도로 등급, 복구난이도
- 5×5 매트릭스로 High/Medium/Low 구분 후 GIS 시각화

### 5단계: 생애주기 비용 분석 (LCCA)

각 자산별로 **유지보수 vs 부분갱생 vs 전면교체** 시나리오의 LCC 비교.
- 할인율 적용한 현재가치(NPV)로 환산
- 잔존수명, 갱생기술 적용성, 미래 수요변화 고려

### 6단계: 최적 투자계획 수립

- **단기계획(1~5년)**: 긴급·고위험 자산 교체
- **중기계획(5~10년)**: 블록 단위 정비사업
- **장기계획(10~30년)**: 권역 단위 갱신 마스터플랜

### 7단계: 사업 시행 및 모니터링

실시간 성과지표(KPI) 추적: 유수율, 누수율, 파손률, 수질 적합률, 민원 건수

### 8단계: 성과평가 및 환류 (PDCA)

연간 단위 목표 대비 실적 평가, 차기 계획 반영 (ISO 55001 지속적 개선 원칙)

### 절차 흐름도

```
[1.계획수립] → [2.DB구축] → [3.상태평가] → [4.위험도평가]
                                              ↓
[8.성과평가] ← [7.시행·모니터링] ← [6.투자계획] ← [5.LCCA]
      ↑___________________________________________|
              (PDCA 환류)
```

## 3. 스마트 통합운영과의 접목 방안

### 핵심 개념

전통적 자산관리: 5~10년 주기 정기조사 데이터 사용
스마트 통합운영 접목: **상시 수집되는 운영데이터가 자산상태 평가의 실시간 입력자료**

```
[기존]  현장조사(5년주기) → 상태평가 → 투자계획
[접목]  IoT/SCADA(실시간) → AI 상태진단 → 동적 위험도 → 적응형 투자계획
```

### 영역별 접목 방안

#### 1. 실시간 상태 모니터링 → 동적 상태평가

- 야간최소유량(MNF) 급증 → 해당 블록 노후관 LoF 가중치 자동 상향
- 수압 변동성 증가 → 펌프·감압밸브 열화 신호
- 잔류염소 저하 → 관 내부 부식·생물막 진행 지표

#### 2. 디지털 트윈 기반 시나리오 분석

- EPANET 관망모델을 SCADA 실측치로 실시간 보정(Real-time Calibration)
- 가상 시나리오, 사고 시뮬레이션, What-if 분석
- `WNTR`(Water Network Tool for Resilience) Python 라이브러리 활용

#### 3. AI 기반 파손 예측 (Predictive Maintenance)

- **LSTM**: 시계열 압력·유량 패턴으로 파손 전조 탐지
- **XGBoost**: 관 속성 + 운영이력으로 향후 1년 파손확률 예측
- **Isolation Forest**: 비정상 운영구간 자동 알림

→ 사후대응(Reactive) → 예방정비(Preventive) → **예지정비(Predictive)** 전환

#### 4. 통합 대시보드 (Single Pane of Glass)

- 실시간 레이어: 유수율, 수압, 수질, 가동상태
- 자산 레이어: 위험도 맵, 잔존수명, 누적 파손이력
- 의사결정 레이어: 투자우선순위, LCCA 결과, 예산집행률

#### 5. 작업관리 시스템 연동 (CMMS 통합)

- 보수공사 완료 → 자산이력 자동갱신 → 잔존수명 재계산
- 민원 접수 → 자산 위치 자동매칭 → 우선순위 알림
- 정기점검 결과 → 상태등급 자동반영

#### 6. 의사결정 자동화

- "위험도 High + 잔존수명 5년 이하 → 차년도 교체사업 후보 자동등록"
- "동일 구간 3회 이상 파손 → 부분보수 대신 전면교체 추천"
- 강화학습으로 예산 제약 하 최적 투자조합 자동탐색

### 시스템 아키텍처

```
[현장계층]   유량계·압력계·수질센서·CCTV
                    ↓ (LoRa/NB-IoT/5G)
[수집계층]   SCADA, IoT 게이트웨이, DCU
                    ↓ (MQTT/OPC-UA)
[데이터계층] 시계열DB(InfluxDB) + 공간DB(PostGIS) + 자산DB(Oracle)
                    ↓ (REST API/Kafka)
[분석계층]   EPANET·WNTR + ML모델(scikit-learn/PyTorch)
                    ↓
[응용계층]   통합 대시보드 + CMMS + 자산관리 모듈
                    ↓
[의사결정]   투자계획·운영전략·비상대응
```

### 단계적 도입 로드맵

1. **1단계(기반조성)**: 자산 DB 정비 + 주요지점 IoT 설치
2. **2단계(연계구축)**: SCADA-자산DB API 연동 + 통합대시보드
3. **3단계(분석고도화)**: 디지털 트윈 + AI 예측모델
4. **4단계(자율운영)**: 의사결정 자동화 + 강화학습 최적화

## 4. K-water SWM 사업 상세

### 사업 정의

**SWM (Smart Water Management)**: 수자원 중심의 물관리 관점을 물 흐름 전 영역으로 확장하며, 스마트 워터 그리드 기반 설계·운영을 통해 물의 안정성·안전성·효율성을 확보하는 통합 물관리 모델.

**광역상수도 SWM**: 수돗물 생산과 공급 전 과정을 AI·ICT 기반으로 스마트화하여 사고 예방 및 신속한 사고대응, 과학적 운영관리로 수돗물 안전관리를 강화하고, 친환경 저탄소 수돗물 생산·공급체계를 구축하는 사업.

### 사업 추진 연혁

| 단계 | 시기 | 주요 내용 |
|------|------|----------|
| 1단계: SWC 시범사업 | 2014~2016 | 파주시 시범사업, 음용률 1% → 36.3%, 만족도 80.7% → 93.8% |
| 2단계: 광역단위 확장 | 2017~2021 | 동두천·양주·정읍·고령·나주 등 21개 지자체 확대 |
| 3단계: 환경부 스마트상수도 관리체계 | 2020~2022 | 161개 지자체, 1조 3,500억 원 |
| 4단계: AI 정수장 시범사업 | 2020~2022 | 화성AI정수장 (글로벌 등대공장 인증) |

### 환경부 스마트상수도 관리체계 4대 영역

1. **수질관리**: 재염소설비, 정밀여과장치
2. **수질감시**: 자동수질측정장치
3. **위기대응**: 자동드레인, 관 세척
4. **물관리**: 스마트미터

### 주요 적용 기술

- 관세척, 정밀여과장치
- 공급과정 수질계측기
- 지능형 관로인식체계
- 자동 배수장치(드레인)
- 지능형 계량기(스마트 미터링)
- 음파 기반 누수탐지

### SWNM (해외수출형 솔루션)

**해외 적용 사례**:
- 인도네시아 스마랑시: 급수인구 19.1만 명, 1,278억 원
- 인도네시아 덴파사르: 급수인구 약 3만 명, 53.2억 원

**효과**: 관로 1km 당 연간 7,000㎥ 누수 저감

### SWM 3S 아키텍처

```
[Device]   IoT센서·스마트미터·자동드레인·수질측정기
              ↓
[Solution] 예측분석·관망해석·AI운영·자산관리
              ↓
[Service]  워터코디·워터닥터·실시간 수질공개·고객앱
```

### 화성AI정수장 핵심 기술

- 생활용수 일 최대 26만㎥ + 공업용수 22만㎥ 공급
- 빅데이터·AI 기반 정수처리공정 자율운영
- 에너지 관리, 설비 상태 예측 기반 사전 예방정비
- 지능형 영상감시
- 세계경제포럼 글로벌 등대공장(Global Lighthouse Network) 인증

## 5. 제139회 상하수도기술사 시험문제 분석

### 상수도 유지관리 핵심 출제 영역

#### 1교시 단답형 핵심

| 문제 | 키워드 | 유지관리 활용도 |
|------|--------|---------------|
| 2번 | 펌프계 수격작용과 대책 | 가압장·송수펌프 운영 핵심 |
| 6번 | TOC (총유기탄소) | 먹는물 수질기준 강화 |
| 7번 | 가압수확산에 의한 혼화 | 정수처리 운영 최적화 |
| 13번 | 단계시험(Step Test) | ⭐ 유수율 제고 핵심 기술 |

#### 4교시 핵심 문제 (★★ 최우선)

**3번 문제**: 상수도 자산관리 시스템 7단계 + 스마트 통합 운영 접목 방안
- 본 문서의 2장, 3장 내용이 직접 답안 구성에 활용 가능

**1번 문제**: 수도시설 유충사고 단계별 대책
- 예방(원수관리) → 감시(여과지) → 대응(자동드레인) → 사후(역학조사) 4단계

### 시험문제가 보여주는 정책 방향

1. **"스마트"와 "자산관리"의 통합이 표준화**
2. **"사고 대응형"에서 "예방 예측형"으로 전환**
3. **노후 인프라 정비가 향후 10년 핵심 사업**
4. **단계시험·블록시스템이 유수율 사업의 기본기**
5. **탄소중립·에너지 절감이 새로운 평가축**
6. **수질 안전이 최우선 가치로 격상**

## 6. AI의 상수도·토목 분야 적용

### 상수도 분야 적용 영역

#### 1. 누수 탐지 및 예측

- **음향 기반**: CNN으로 누수음 패턴 학습 (정확도 95% 이상)
- **시계열 기반**: LSTM/Transformer로 유량·압력 패턴 학습

```python
# 야간최소유량 이상탐지 예시
from sklearn.ensemble import IsolationForest
import pandas as pd

mnf_data = df[df['time'].between('02:00','04:00')]
model = IsolationForest(contamination=0.05)
mnf_data['anomaly'] = model.fit_predict(mnf_data[['flow','pressure']])
# anomaly=-1 인 시점이 누수 발생 가능 구간
```

## 7. 음향 기반 누수탐지 연구 사례

### 국내 사례

#### K-water 스마트관망관리 누수탐지 시스템

지방상수도 스마트관리체계의 핵심 기술로 **음파 기반 누수감지** 운영 중. 자동수질측정장치·자동드레인·스마트미터링과 통합 운영.

#### K-water AI 경진대회 (2024, 4회차)

- **주제**: 상수도 관망 이상 감지 AI 알고리즘 개발
- **목표**: 이상 시점과 누수 발생 구간 정확 탐지
- **데이터**: 분 단위 압력계 시계열 데이터
- **활용**: 상수관망 디지털트윈, Water-Net 사내 시스템 내재화
- **플랫폼**: DACON

#### 유솔(USOL) - 상용 솔루션

- 대구 국가물산업클러스터 입주 기업
- 매일 30만개 데이터 학습 (전기율·기계음·기타음 구분)
- 하루 10회 측정 + 20일 평균변화량 + 표준편차 기반 AI 학습

### 국제 학술연구 최신 트렌드

#### 주요 접근법 발전 단계

```
1세대: 단순 신호처리 (FFT, 필터링)
2세대: 전통 머신러닝 (SVM, Random Forest, AdaBoost)
3세대: CNN 기반 딥러닝 (2D 스펙트로그램 입력)
4세대: Transformer, Contrastive Learning (현재)
5세대: PINN, Federated Learning (연구 진행 중)
```

#### 주요 모델 성능 비교

| 모델 | 정확도 | 특징 |
|------|-------|------|
| Random Forest | 81.2% | 특징 기반 |
| Standard CNN | 82.9% | 스펙트로그램 기반 |
| FI-CNN (2025) | 90.4% | 통계특징+원신호 결합 |
| RCT-Net | 97% (AUC 99.7%) | Transformer + 등급 분류 |
| SA Water 실증 (호주) | 92.44% | 도시 전역 실증 |

#### 최신 연구 주제

1. **CNN 기반 STFT/웨이블릿 변환** - 정확도 95~99%
2. **RCT-Net** - 누수 등급(경미·중간·심각) 분류까지 가능
3. **FI-CNN (2025)** - 파이프 형상·센서 위치 변화에도 강건
4. **Contrastive Learning** - 라벨 데이터 부족 해결
5. **Two-stage Temporal Segmentation (2025)** - 잡음 강건성 강화

### 해외 실증사업

- **호주 SA Water (Adelaide)**: CNN 기반 모델로 92.44% 정확도 달성
- 도시 전역 스마트 워터 네트워크(SWN) 운영

### 연구과제 미해결 이슈

1. **라벨 데이터 부족** → Contrastive Learning, Self-Supervised Learning
2. **도메인 일반화** → 다양한 환경·관망에 강건한 모델
3. **설명가능성(XAI)** → 공공사업 적용 필수
4. **엣지 컴퓨팅** → 센서 단말 실시간 추론 (TinyML)
5. **멀티모달 융합** → 음향 + 유량 + 압력 + 영상 + 위성
